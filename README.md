# models
Contain all models and their properties

## How to use

1. retrieve model list
https://raw.githubusercontent.com/activebook/models/main/list.json

2. use model name to retrieve model detail
https://raw.githubusercontent.com/activebook/models/main/models/[model_name].json
e.g.
https://raw.githubusercontent.com/activebook/models/main/models/minimax_minimax-m2.5.json
